# 🎵 背景音乐建议方案

## 🎼 音乐风格定位

基于**武陵城风格**（新中式科幻美学），推荐以下音乐类型：

### ✅ 推荐风格
1. **古风电子乐** - 传统乐器 + 电子合成器
2. **氛围音乐 (Ambient)** - 轻柔、不干扰阅读
3. **Lo-fi Hip Hop** - 轻松、有节奏感
4. **新世纪音乐 (New Age)** - 空灵、治愈
5. **后摇 (Post-Rock)** - 情感丰富、层层递进

## 🎚️ 使用建议

### 音量控制
```javascript
// 推荐音量范围
const backgroundVolume = 0.3; // 30% 音量（不干扰阅读）
```

### 播放策略
1. **默认静音**
   - 用户主动开启
   - 避免突然出声

2. **渐入渐出**
   ```javascript
   // 淡入时间：2 秒
   fadeIn: 2000ms
   
   // 淡出时间：1 秒
   fadeOut: 1000ms
   ```

3. **循环播放**
   - 单曲循环 or 随机播放
   - 避免突兀切换

### 场景匹配
| 页面类型 | 推荐 BPM | 情绪 |

| 首页 | 80-100 | 平静、欢迎 |
| 相册页 | 60-80 | 舒缓、回忆 |
| 视频页 | 100-120 | 动感、活力 |
| 感悟页 | 50-70 | 沉思、内省 |
| 关于页 | 70-90 | 温暖、亲切 |

## 🔧 技术实现方案

### React 组件示例
```jsx
import { useState, useEffect } from 'react';

function BackgroundMusic() {
  const [isPlaying, setIsPlaying] = useState(false);
  const [volume, setVolume] = useState(0.3);

  useEffect(() => {
    const audio = new Audio('/music/bgm.mp3');
    audio.loop = true;
    audio.volume = volume;

    if (isPlaying) {
      audio.play();
    }

    return () => {
      audio.pause();
      audio.currentTime = 0;
    };
  }, [isPlaying, volume]);

  return (
    <button onClick={() => setIsPlaying(!isPlaying)}>
      {isPlaying ? '🔊 关闭音乐' : '🔇 播放音乐'}
    </button>
  );
}
```

## ⚠️ 注意事项

### 版权问题
- ✅ 使用 CC0 / CC-BY 协议音乐
- ✅ 标注作者和来源
- ❌ 避免未授权商业音乐
- ❌ 不要直接嵌入网易云/QQ 音乐

### 性能优化
- 压缩音频文件（MP3 128kbps 足够）
- 懒加载（需要时再加载）
- 提供开关（尊重用户选择）

### 用户体验
- 默认静音（关键！）
- 明显的控制按钮
- 记忆用户偏好（localStorage）
